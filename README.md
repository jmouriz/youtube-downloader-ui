youtube-downloader-ui
=====================

A frontend for youtube-dl script.

Little UI frontend for youtube-dl for download Youtube videos. GTK+ 2 & 3 version.

![screenshot](https://dl.dropboxusercontent.com/u/36581494/captura-9.png "Screenshot")

This is a usage demo of GLib g_spawn_async_with_pipes function and how to parse output with GRegex.
